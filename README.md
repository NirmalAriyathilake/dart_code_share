# Observable Flutter

Dart code sharing project based on [Observable Flutter](https://www.youtube.com/playlist?list=PLjxrf2q8roU1GHtc2FCHoEZr_v-LqnTZX) playlist from Flutter youtube channel.

### Tech: 
    * Dart
    * Flutter
    * Postgres DB
    * Dart Frog - Server
    * Stormberry - Code generation package
