# Dart code share

Dart code sharing project to demonstrate a Flutter client & a Dart server sharing same code through shared packages.

## Tech: 
    * Dart
    * Flutter
    * Postgres DB
    * Dart Frog - Server
    * Stormberry - Code generation package
